== Description ==

With this plugin you can see how people were referred to your site, ideal if you want to see how people are finding your blog.

All you need to do is select how many referrers you wish to show and the title of the widget.

== Installation ==

1. Upload `seereferrers-plugin.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Edit the Referrer WP Settings in your Administration Panel
4. Enable the Widget through the Appearance --> Widgets tab.
